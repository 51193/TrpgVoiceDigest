# 人物卡示例

将人物卡文件（`.md` 格式）放入 Campaign 目录下的 `character_cards/` 文件夹中，系统将自动读取并附加到所有 LLM 提示词中（精炼、说话人识别、一致性表维护）。

## 格式说明

人物卡采用 Markdown 格式，重点关注 **信息密度** 而非叙事文学性。每张人物卡应尽可能简洁地包含以下信息：

- **种族**：角色的种族/物种
- **职业**：角色的职业/定位
- **关键特质**：最重要的性格、行为特征
- **核心能力**：关键技能和能力
- **外观**：可辨识的外貌特征
- **背景**：简要背景（一两句话即可）
- **人际关系**：与其他角色/NPC 的关系

## 文件命名

建议使用角色名称命名文件（如 `梅林.md`），文件名不影响使用，仅用于人工管理。
