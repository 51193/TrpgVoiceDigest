# 输出协议：任务

每行一个操作。格式：`task <action> [number] "text"`

## 支持的操作

| 操作       | 格式                     | 说明                           |
|:---------|:-----------------------|:-----------------------------|
| add      | `task add "任务文本"`      | 在末尾追加一个新活跃任务                 |
| insert   | `task add <编号> "任务文本"` | 在指定编号之后插入一个新任务               |
| edit     | `task edit <编号> "新文本"` | 修改指定编号的任务描述                  |
| remove   | `task remove <编号>`     | 删除指定编号的活跃任务（仅活跃任务，已完成任务不可删除） |
| complete | `task complete <编号>`   | 将活跃任务标记为成功完成，移至已完成区          |
| fail     | `task fail <编号>`       | 将活跃任务标记为失败/放弃，移至已完成区         |

## 格式约束

- 每行一组完整操作，不换行
- 文本内容中的双引号用 `\"` 转义
- 无任务操作时不输出任务相关行（只输出故事进展行或 EMPTY）
- **禁止**输出解释性文字、Markdown 代码块包裹或 JSON 包裹
- **禁止**输出 tags 字段
