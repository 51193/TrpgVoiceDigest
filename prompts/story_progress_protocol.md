# 输出协议

每行一个操作。格式：`story <action> [number] "text"`

## 支持的操作

| 操作     | 格式                      | 说明             |
|:-------|:------------------------|:---------------|
| add    | `story add "事件文本"`      | 在末尾追加一个新故事事件   |
| add at | `story add <编号> "事件文本"` | 在指定编号之后插入一个新事件 |
| edit   | `story edit <编号> "新文本"` | 修改指定编号的事件内容    |
| remove | `story remove <编号>`     | 删除指定编号的事件      |

## 格式约束

- 每行一组完整操作，不换行
- 文本内容中的双引号用 `\"` 转义
- 无任何操作时输出单行 `EMPTY`
- **禁止**输出解释性文字、Markdown 代码块包裹或 JSON 包裹
- **禁止**输出 tags 字段
